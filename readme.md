# 雨轩工具箱

一个面向《魔兽世界》正式服的综合型插件，提供地图辅助、界面美化、战斗辅助和部分职业功能入口。

## 当前版本

- `1.1.7`

## 主要功能

- 常用功能：快捷导航、快捷频道、游戏条、任务工具
- 界面美化：界面整理、鼠标提示、鼠标拖尾、背包文字、属性与货币显示
- 探索与团队：地图信息、狩猎助手、团队工具通报、事件追踪、团队标记
- 战斗与职业：快速焦点、饰品监控、技能监控
- 通用：外观设置、布局导入、配置管理

## 使用方式

- 输入 `/yxs` 打开设置窗口
- 输入 `/yxs log` 打开更新记录

## 这次更新

- 新增技能监控工作台和独立配置管理
- 新增团队工具通报，可自动按队伍或团队频道发送职业工具技能提示
- 修复全局颜色选择器导致的配置初始化和颜色丢失问题
- 修复技能监控配置在重载后看起来未保存的问题
