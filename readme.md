# 雨轩工具箱

一个面向《魔兽世界》正式服的综合型插件，提供地图辅助、界面美化、战斗辅助和部分职业功能入口。

## 当前版本

- `1.1.8`

## 主要功能

- 常用功能：快捷导航、快捷频道、游戏条、任务工具、宏伟宝库低保查询
- 界面美化：聊天输入优化、鼠标提示、鼠标拖尾、背包文字、属性与货币显示
- 探索与团队：地图信息、狩猎助手、团队工具通报、事件追踪、团队标记
- 战斗与职业：快速焦点、饰品监控、技能监控
- 通用：外观设置、布局导入、配置管理、更新记录

## 使用方式

- 输入 `/yxs` 打开设置窗口
- 输入 `/yxs log` 打开更新记录
- 输入 `/yxs db` 打开低保查询

## 1.1.8 更新

- 新增狩猎进度条，并继续调整阶段百分比和展示逻辑，让野外进度更直观。
- 新增 EnhanceQoL 联动设置与运行时接入，方便和现有增强功能一起使用。
- 新增宏伟宝库低保查询页、`/yxs db` 快捷命令和游戏条宏伟宝库按钮。
- 宏伟宝库显示改成更紧凑的单行进度格式，团本奖励预览也修正为优先读取当前奖励装等。
- 移除法师碎冰监控模块，并清理一批死代码、冗余逻辑和不必要的模块加载。
- 针对插件长时间运行时的内存上涨问题，补上更积极的 Lua 增量 GC，并将冷却管理器隐藏扫描改为按需启动。
